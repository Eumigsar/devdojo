# Sistema de Conteúdo

## Vocabulário Vivo

Cada termo deve ter:

- `id`
- `hanzi`
- `pinyin`
- `portuguese`
- `category`
- `realm`
- `discoveredIn`
- `example`

## Missões

Cada missão deve ter:

- tipo: idioma, físico, cultura, taolu, história
- recompensa de XP
- recompensa de moeda
- palavras desbloqueadas
- condição de conclusão

## Revisão Invisível

Em vez de flashcards diretos, revisar através de:

- falas de NPC
- placas
- portas bloqueadas
- escolhas de diálogo
- treino marcial guiado

Exemplo:

> O portão mostra: 前 / 后 / 左 / 右.  
> O mestre diz: “Vá para a esquerda.”  
> O jogador precisa escolher 左.
