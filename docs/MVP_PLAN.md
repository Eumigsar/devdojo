# Plano de MVP — Protótipo Jogável

## Fase 1 — Organização

- Subir projeto no GitHub
- Manter Vite + Babylon.js
- Adicionar `src/data`, `src/systems` e `docs`
- Garantir que `npm run dev` funcione

## Fase 2 — Conteúdo Vivo

- Conectar `vocabulary.ts` à HUD
- Mostrar palavra descoberta quando o jogador interagir com mestre/objeto
- Salvar palavras descobertas no `localStorage`

## Fase 3 — Missões

- Exibir missão diária
- Completar missão manualmente no MVP
- Recompensar com XP, moeda e vocabulário

## Fase 4 — Mascote

- Criar card do mascote na HUD
- Evoluir por vocabulário + streak + treino

## Fase 5 — Taolu

- Criar tela/lista da primeira forma
- Cadastrar etapas, aplicações e vocabulário associado

## Fase 6 — Polimento

- Melhorar UI visual novel
- Adicionar efeitos de partículas/Qi
- Adicionar sons e música ambiente
